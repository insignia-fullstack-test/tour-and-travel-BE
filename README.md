# TOUR KUY 

TOUR KUY is an API-based service for travel package

## Requirements

- Node.js 16+

## Installation

```
$ npm install
```

You'll need to fill the contents of `.env` file for your local database Postgresql.

## Usage


## run sequelize

```
sequelize db:create
sequelize db:migrate
sequelize db:seed:all
```

Run the server

```
$ npm run dev
```
